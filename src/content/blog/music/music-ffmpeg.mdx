查看mp3的信息
```
 ffprobe -v error -show_entries stream=codec_type,codec_name,sample_rate,bit_rate -of default=noprint_wrappers=1 .\tears-on-the-dancefloor-stocktune.mp3
```

可能值
```
codec_name=mp3
codec_type=audio
sample_rate=48000
bit_rate=128000
```
